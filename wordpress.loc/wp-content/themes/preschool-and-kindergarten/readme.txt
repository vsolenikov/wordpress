== Description ==

Preschool and Kindergarten is a educational WordPress theme suitable for Kindergarten, Schools, Elementary, Primary Schools, Secondary School, Universities, Academics and Educational institutions. Preschool and Kindergarten is also suitable for education and children activities. It is mobile responsive and translation ready.


== Installation ==
        
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Documentation ==
    * Home Page Setup
    * To set up the homepage you must first create a new page.
        => Create a Home Page
        => Go to Pages > Add New.
        => Enter the Page Title for the Page. You do not require to enter anything in the content section for creating a home page.
        => Select the Homepage Template from the Page Attributes section.
        => Click Publish.
        
    * Configuring the front Page display setting
        => Go to Appearance>Customize > Default Settings> Static Front Page.
        => Select A static page under Front Page displays
        => Choose the page you just created with homepage template as Front page.
        => Click Save & Publish.

    * Contact Info
        => Go to Appearance> Customize>Home Page Settings>Contact Info 
        => Enter Email Address and Phone Number.
        => Click Save & Publish.

    * Slider Settings
        => Go to Appearance> Customize>Home Page Settings>Slider Settings
        => Check Enable Slider section for it to appear on Homepage.
        => Check Enable Slider to Auto Transition for the slides to transition automatically (auto-scroll) without a click.
        => Check Enable Slider Loop to continue the slide transition in the loop
        => Check Enable Slider Control to allow manual navigation for paging control of each slide.
        => Check Enable Slider Caption to show the post title to appear on the slider post image
        => Select the slider animation type to “fade” or “slide”
        => Enter the Readmore text to set the text of the button
        => Enter Slider Speed: Set the Interval (in milliseconds) at which the slider should change posts (if automatic transition is enabled). Default: 7000 (7 seconds)
        => Enter Animation Speed: Set the speed of animations, in milliseconds. Default: 600
        => Select the category of posts which you want to display in the slider. (Featured images of the posts of the selected category will be displayed as slider images. Post Title will appear as Slider Title and Excerpt of the Post will appear on slider as slider description )
            
            * Featured images of posts should be at least 1920 px in width and 500 px in the height for displaying slider images properly.
            
            * Use regenerate thumbnail plugin to crop all the images if you have already uploaded the Images.
        
        => Click Save & Publish.
        
    * About Section
       Note: Note before you start setting up the About section. Make a page with default template with the title and content that needs to be displayed under About us Section.

        => Go to Appearance> Customize>Homepage Settings> About Section  
        => Check Enable About Section to display it in the Homepage.
        => Choose the page for About Section.
            * The Page title of the selected page will appear as the section title.
            * The featured image and excerpt of the selected page will be displayed under About  Section.
        => Click Save and Publish.

    * Lesson Section
        => Go to Appearance> Customize>Homepage Settings> Lesson Section  
        => Check Enable Lesson Section to display it in the Homepage.
        => Enter Section Title and Section description.
        => Choose Post One, Post Two and Post Three
            * The title and the content of the selected post will be displayed as the Lesson title and content
            * The featured Image of the selected Post will be displayed as the Lesson Image.
        => Click Save & Publish.
    
    * Services Section
        => Go to Appearance> Customize>Home Page Settings> Services Section.
        => Check Enable Service Section for it to appear on Homepage.
        => Enter the section title and section description.
        => Select a Post for the Services section.
            * The content of the Post will be displayed as the section content.  The listed items in the content will appear as the services.
            * The Featured Image of the selected Post will be displayed under the section.
        => Click Save & Publish. 
    
    * Promotional Block Section
        => Go to Appearance> Customize>Home Page Settings> Promotional Block Section.
        => Check Enable Promotional Block  Section for it to appear on Homepage.
        => Enter the section title and section description.
        => Enter Button Label and Button Link. 
        => Click Save & Publish.

    * Program Section
        => Go to Appearance> Customize>Homepage Settings> Program Section
        => Check Enable Program Section to display it in the Homepage.
        => Enter Section Title and Section description.
        => Choose Post One, Post Two and Post Three
            * The title and the excerpt of the selected post will be displayed as the Program title and content
            * The featured Image of the selected Post will be displayed as the Program Image.
        => Click Save & Publish.

    * Testimonial Section:
        => Go to Appearance> Customize>Home Page Settings> Testimonial Section.
        => Check Enable Testimonial Section for it to appear on Homepage.
        => Enter Section Title and Section Description.
        => Choose category  for the section.
            The Featured Image of the Posts of the selected category will be displayed under the section as client image.
            The Title of the posts under the selected category will appear as client name .
            The excerpt of the posts of the selected category  will be displayed as client  address.
            The content of the posts of the selected category will be displayed as the testimonial.
        => Click Save & Publish.
    
    * Team Section
        => Go to Appearance> Customize>Homepage Settings> Team Section  
        => Check Enable Team Section to display it in the Homepage.
        => Enter Section Title and Section description.
        => Choose Post One, Post Two and Post Three
            * The Featured Image of the Posts of the selected category will be displayed under the section as team member’s image.
            * The Title of the posts under the selected category will appear as team member’s name .
            * The excerpt of the posts of the selected category  will be displayed as designation.
        => Click Save & Publish.
    
    * News Section
        => Go to Appearance> Customize>Home Page Settings> News Section.
        => Check Enable News Section for it to appear on Homepage.
        => Enter Section Title and Section description.
        => Choose Category for the section
            * The latest three posts from the selected category will be displayed under the section.
            * The title and excerpt of the posts of the selected category will be displayed as  the news title and content.
            * The posts published date will appear beside the news post.
        => Click Save & Publish
